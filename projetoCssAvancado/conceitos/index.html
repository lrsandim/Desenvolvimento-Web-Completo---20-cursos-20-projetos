seletor universal

*{}

seleciona todos os elementos da página

seletor de texto

H1, h2, h3 {}

seleciona os elementos h1, h2 e h3

seletor de classe

.verde p.verde

.verde seleciona qualquer elemento cujo atributo class tem o valor "verde"

seleciona somente elemento p cujo atributo class tem valor "verde"

seletordeid

#

seleciona o elemento cujo atributo id tem valor cabecalho

seletor de filho

li>a

seleciona quaisquer elemento -a- que são filhos de um elemento -li- (mas não outros elementos -a- na página

seletor de descendente

p a {}

seleciona quaisquer elementos -a- que são filhos de um elemento -p-, mesmo e houver outros elementos aninhados entre ele

seletor de irmao adjacente

h1+p

seleciona o primeiro elemento -p- depois de qualquer elemento -h1- (mas não outros elememtos -p-)

seletor de irmao geral

h1~p

se houvesse dois elementos -p- que fossem irmãos de um elemento -h1-, essa regra se aplicaria ao dois
